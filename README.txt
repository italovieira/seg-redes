Servidor
Primeiramente inicie o servidor com python3 server.py
Então, o primeiro usuário conecta-se executando python3 client.py e informando o
endereço do servidor iniciado.
A mesma coisa para o segudo usuário.

Após informar o nome um do outro, já podem se comunicar entre si usando a
criptografia padrão.

Mas também é possível alterá-la enviando no chat o comando '/crypt [sdes | rc4] key'
